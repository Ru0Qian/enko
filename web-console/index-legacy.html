Enko Web Console

把加固流程搬到网页端，本地就能直接开跑。

这版不是展示图，而是给你本机直接用的控制台。 可以选功能、生成命令、启动本地加固任务、查看日志，再把 report.json 拉进来审计。

运行时门闩 6 + native-core
命令模式 Flutter
当前策略 block / strict
